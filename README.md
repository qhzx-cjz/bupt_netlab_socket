<h1>北京邮电大学_2023级_计算机网络_实验二_基于TCP的文件传输与socket程序设计</h1>

<h2>启动服务端</h2>

```bash
python ./socket_server.py
```

<h2>启动用户端</h2>

```bash
python ./socket_client.py
```

<h2>连接服务端</h2>

```bash
Enter server IP and port: localhost:8888
```
<h2>获取文件命令</h2>

```bash
GET test.txt
```
<h2>退出命令</h2>

```bash
exit
```

<h2>关闭服务</h2>

```bash
close
```
<h2>不支持SSL安全连接</h2>
